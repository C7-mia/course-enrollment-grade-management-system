# Course Enrollment and Grade Management System

## Description
This project is a Java-based university course enrollment and grade management system developed using Object-Oriented Programming principles.

## Features
- Add courses
- Enroll students
- Assign grades
- Calculate overall grades
- Track total enrolled students

## Technologies Used
- Java
- GitHub Codespaces
- GitHub
